# RevTrack — Revenue Leakage Prevention Platform

RevTrack is a professional-grade platform designed to help service businesses identify and recover lost revenue. By connecting time logging, project management, and invoicing into a single, intelligent workflow, RevTrack ensures that every billable minute is captured and every invoice is tracked to completion.

## 🚀 Key Features

### 📊 Revenue Intelligence Dashboard
- **Leakage Detection**: Real-time identification of unbilled hours and aging invoices.
- **Visual Analytics**: Monthly revenue trends and client-based revenue distribution charts.
- **Actionable Insights**: AI-ready suggestion engine that prompts users to bill for completed work or follow up on overdue payments.

### 👥 Client & Project Management
- **Centralized Directory**: Manage client contact information, billing currencies, and default rates.
- **Project Tracking**: Organize work by project with specific budget tracking and task management.
- **Status Monitoring**: Track active vs. archived clients and projects at a glance.

### ⏱️ Time & Work Logging
- **Billable Tracking**: Log hours against specific projects with detailed descriptions.
- **Real-time Sync**: Time logs are immediately available for invoicing once recorded.

### 🧾 Intelligent Invoicing
- **One-Click Generation**: Automatically pull unbilled time logs into professional invoices.
- **Invoice Lifecycle**: Track invoices from Draft to Sent, Paid, or Overdue.
- **Aging Reports**: Visibility into how long revenue has been sitting uncollected.

### 💳 Payment Tracking
- **Multi-method Support**: Record payments via Bank Transfer, Card, Cash, or other methods.
- **Payment History**: Maintain a clear audit trail of all transactions linked to specific invoices.
- **Automatic Reconciliation**: Invoice statuses update automatically as payments are recorded.

## 🛠️ Technical Stack

- **Frontend**: React 19, Vite, TypeScript
- **Styling**: Tailwind CSS 4 (Professional Polish Theme)
- **UI Components**: shadcn/ui, Lucide React, Framer Motion
- **Database & Auth**: Firebase (Firestore & Authentication)
- **Charts**: Recharts
- **State Management**: React Hooks & Context

## 🎨 Design Philosophy: "Professional Polish"

RevTrack features a custom-crafted interface designed for clarity and precision:
- **Slate & Indigo Palette**: A deep, professional color scheme that emphasizes data over decoration.
- **Technical Typography**: Using Inter for UI and JetBrains Mono for financial figures to ensure maximum legibility.
- **Responsive Layout**: Desktop-first precision with a fluid, sectioned navigation system.

## 🏁 Getting Started

1. **Authentication**: Sign in using your Google account to access your private workspace.
2. **Setup Clients**: Add your clients and set their default billing rates.
3. **Log Work**: Start logging time against projects as work is completed.
4. **Bill**: Use the Invoices tab to generate bills for your unbilled work.
5. **Track**: Monitor the Dashboard for revenue leaks and follow up on overdue items.
